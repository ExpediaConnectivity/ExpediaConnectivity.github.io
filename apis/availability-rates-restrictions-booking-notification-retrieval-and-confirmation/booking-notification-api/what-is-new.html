What is New

Changes to Expedia Collect Notifications - Mexico and beyond

(Last Updated: 2020-09-07)

Expedia Group is updating its Expedia Collect (EC) model to simplify how it operates and to remain relevant and current in local markets.

For more information, please refer to this blog post.

The additional elements & attributes in the API Definition are as follows: PaymentMethod, Payment (@Amount, @CurrencyCode, @Model), PaymentCard (@CardActivation), BaseRateType

Expedia Collect – Gross payment model with EVC

Daily sell rate/LAR: $200*

EG compensation (20%): $200 (sell) x 20% (EG comp) = $40

Daily net rate: $200 (sell) - $40 (EG comp) = $160

Taxes & fees on base (sell/LAR) (16%): $32*

  • On net: 16% x $160 = $25.60
  • On EG comp: 16% x $40 = $6.4

Total booking amount: $232*

If the property is liable for payment of all taxes to the government:

Payment amount: $200 (sell) + $32 (total taxes/fees) = $232*

Amount EG will invoice for: $40 (EG comp)

If EG is liable for payment of tax on compensation to the government:

Payment amount: $200 (sell) + $32 (total taxes/fees) = $232*

Amount EG will invoice for: $40 (EG comp) + $6.40 (taxes/fees on EG comp) = $46.40

*Bolded values are sent in the schema

XML Snippet:

    <Rate EffectiveDate="2020-07-22" ExpireDate="2020-07-23" RateTimeUnit="Day" UnitMultiplier="1">
        <Base AmountBeforeTax="200.00" CurrencyCode="USD"/>

            <Fee Amount="0.00" Code="1" CurrencyCode="USD" TaxInclusive="false" Type="Exclusive"/>
        </Fees>
    </Rate>
</Rates>

    <GuestCount AgeQualifyingCode="10" Count="2"/>
</GuestCounts>
<TimeSpan End="2020-07-24" Start="2020-07-22"/>
<PaymentMethod>VirtualCard</PaymentMethod>
<Payment Amount="232.00" CurrencyCode="CAD" model="Gross">
    <Guarantee>
        <GuaranteesAccepted>
            <GuaranteeAccepted>
                <PaymentCard CardCode="AX" CardNumber="*****" CardType="1" ExpireDate="0725" SeriesCode="*****" CardActivation="YYYY-MM-DDThh:mm:ss[+/-]hh:mm">
                    <CardHolderName>*****</CardHolderName>
                    <Address FormattedInd="false">
                        <AddressLine>*****</AddressLine>
                        <CityName>Seattle</CityName>
                        <PostalCode>98119</PostalCode>
                        <StateProv StateCode="WA"/>
                        <CountryName Code="US"/>
                    </Address>
                </PaymentCard>
            </GuaranteeAccepted>
        </GuaranteesAccepted>
    </Guarantee>
    <Total AmountAfterTax="232.00" CurrencyCode="USD">
        <Taxes Amount="32.00" CurrencyCode="USD">
            <Tax Amount="32.00" Code="27" CurrencyCode="USD" Type="Exclusive"/>
        </Taxes>
    </Total>
    <BaseRateType>Sell</BaseRateType>
